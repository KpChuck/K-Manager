Moving Indicators to the Left

FAQ's Other Roms Help Custom Qs Backgrounds Left Network Indicators

This is a much easier process than you think!

  1. Select the option in K-Manager to move the network indicators to the left.
  2. Tap the yellow build button at the bottom left corner of K-Manager.
  3. Install K-Klock once it's been built.
  4. Open K-Klock in substratum and build the clock that you want, don\'t worry about not being able to find any option for the left indicators!
  5. Reboot or restart SystemUI and watch as your network indicators move to the left.

Important! When installing a clock without left network indicators, make sure that you uninstall this overlay first.
