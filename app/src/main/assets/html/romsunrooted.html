About other Roms

  • This option modifies the xml files necessary to create K-Klock on the go
  • Unfortunately, these xml files are different for each Rom
  • Since I don't have the xml files from every Rom, you'll have to provide these yourself
  • Below is an easy step-by-step guide that anyone should be able to follow

Instructions

  1. Download an apk decompiler

    I would recommend Apk Editor It's faster than using a Pc for everything.

  2. Access your Roms SystemUI.apk

    You're going to need a PC for this part. Make sure you have Adb Drivers installed on your PC (Google how) if you're using Windows. Open up a command terminal and type:

    1. adb pull /system/priv-app/SystemUI/SystemUI.apk
    2. adb push SystemUI.apk /sdcard/
    3. The location and name may differ slightly between devices

    You should now have a file called SystemUI.apk in your internal storage.

  3. Find and decompile your Roms SystemUI.apk

    • Open up Apk Editor
    • Select the Apk from File option.
    • Now search for the SystemUI.apk that you extracted and decompile with Full Edit.

    Now select Files at the bottom of Apk Editor

  4. Exctracting the files

    Navigate to res -> layout and select the following three xml's:

    • keyguard_status_bar.xml
    • status_bar.xml
    • system_icons.xml

    Then use the Extract option at the bottom to extract the xmls to K-Klock/userInput (case-sensitive).

  5. The Fun Part!

    Open up K-Manager and select the Other Roms option as your Rom. You won't have to repeat this process unless those files are deleted or you change Roms.
