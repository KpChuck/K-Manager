About other Roms

  • This option modifies the xml files necessary to create K-Klock on the go
  • Unfortunately, these xml files are different for each Rom
  • Fortunately K-Manager can now decompile your SystemUI.apk and extract those files
  • If you're not rooted you will have to run an ADB command that you can copy to your clipboard if you try building

Contribute

Add your Rom into the list of supported Roms in K-Manager.

Once you have tested that your Other Roms options work, they can be added to K-Manager.

You can also use the Other Roms settings to add extra functionality for your Rom.

Submitting files to be included in K-Manager - Guide

  • Open a filemanager and navigate to /sdcard/K-Klock. You should have a userInput folder there.
  • Zip all the xmls inside your userInput folder. Rename the zip file to your name of your Rom with it's android version. For example 'Paranoid Android Nougat.zip'
  • You can fork K-Manager and make a pull request to submit the files
  • Or just send it to me via email, telegram, xda. If you do it this way, you can ignore the rest of this guide.
  • If you're making a pull request, place the zipped xmls into the app/src/main/assets/romSpecific folder.
  • If you have tested the Dev Options in K-Manager, open the file rom_instructions.xml in app/src/main/res/values and add your Rom to the lists there.
    Make sure that the text there is exactly the same as the name of your zip file without the .zip!
  • Then just open up a pull request on github and I'll merge your files into K-Manager
