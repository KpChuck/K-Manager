Передача файлов для включения в K-Manager - инструкция

  • Откройте K-Manager, в качестве Вашей прошивки выберите Другие прошивки и нажмите на Настройки других прошивок.
  • Нажмите опцию передать файлы в K-Manager, заполните детали Вашей прошивки, откроется предложение передать их мне по электронной почте или через Telegram.
  • Или же, Вы можете сщздать себе ответвление в репозитории K-Manager и сделать pull request для передачи файлов
  • После выполнения вышеописанных действий можете перейти в /sdcard/K-Klock и убедиться, что там есть архив с деталями, которые Вы заполнили.
  • Поместите архив в папку app/src/main/assets/romSpecific.
  • Если Вы работали с опциями разработчика в K-Manager, откройте файл rom_instructions.xml в app/src/main/res/values и добавьте Вашу прошивку в списки.
    Убедитесь, что текст там полностью совпадает с именем Вашего архива без .zip!
  • Затем просто откройте pull request на github и я добавлю Ваши файлы в K-Manager.
